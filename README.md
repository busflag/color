# 🎨 颜色工具包 | Color Toolkit

一个功能全面的现代化颜色工具，集交互式取色、格式转换、配色方案生成与专业对比度检查于一体。为设计师、开发者和任何需要处理颜色的用户打造。

**🌐 在线直达链接：[https://yanse.netlify.app/](https://yanse.netlify.app/)**

---

## ✨ 核心功能

### 1. 🖱️ 交互式画布取色器
- **全彩光谱**：在Canvas绘制的色板中点击，实时获取精准颜色值。
- **多种格式**：同步显示颜色的 **HEX**、**RGB**、**HSL** 值。
- **实时预览**：大面积预览当前选中颜色及其对比色。

### 2. 🔄 智能格式转换
- **双向转换**：在HEX、RGB、HSL三种主流格式间无缝、精准转换。
- **透明度支持**：完整支持RGBA和HSLA格式，满足半透明需求。
- **一键复制**：每种格式旁均设有复制按钮，便捷分享。

### 3. 🎨 专业配色方案生成器
基于色彩理论，一键生成协调配色方案：
- **互补色** - 高对比度，充满活力
- **类似色** - 和谐统一，舒适自然
- **三色组** - 丰富多变，平衡感强
- **分裂互补色** - 高对比度但更柔和
- **四色组（矩形）** - 双互补色，丰富多彩
> **点击任意生成色块，即可将其设为主色。**

### 4. ⚖️ WCAG对比度检查器
- **专业计算**：实时计算前景色与背景色的对比度比率。
- **无障碍标准**：依据 **WCAG 2.1** 标准，清晰标注是否通过 **AA (≥4.5:1)** 和 **AAA (≥7:1)** 等级。
- **设计助手**：确保你的色彩选择对所有用户都具有良好的可读性。

### 5. 💾 个人色板与本地存储
- **色板收藏**：随时将喜爱的颜色保存至个人色板。
- **本地持久化**：所有收藏的颜色通过 `localStorage` 保存在浏览器中，关闭页面也不丢失。
- **便捷管理**：点击色板快速应用，支持删除不需要的颜色。

## 🛠️ 技术栈与实现

- **核心技术**：原生 `JavaScript (ES6+)`， `HTML5 Canvas API`， `CSS3`
- **核心算法**：
  - 自主研发的**颜色空间转换算法**（RGB ↔ HSL）。
  - 基于**色相环**的配色方案计算逻辑。
  - 遵循 **WCAG** 标准的**相对亮度与对比度计算**。
- **数据存储**：利用浏览器 `localStorage` 实现数据持久化，无需后端。
- **UI/UX**：
  - 采用 **CSS Grid** 与 **Flexbox** 构建响应式布局。
  - 使用 **CSS Variables** 动态管理主题色，为深色模式预留支持。
  - 全站交互应用平滑的 **CSS Transitions** 动画。
